# GroundPlane Persistence Policy AI Extension

Pure contract package for AI Platform data classes, memory-default profiles,
store capability descriptors, durable opt-in preflights, and debug redaction
helpers.

This package does not depend on databases, Temporal, provider SDKs, object
stores, vector stores, product repos, or LiveView.

## Persistence Documentation

See `docs/persistence.md` for tiers, defaults, adapters, unsupported selections, config examples, restart claims, durability claims, debug sidecar behavior, redaction guarantees, migration or preflight behavior, and no-bypass scope when applicable.
