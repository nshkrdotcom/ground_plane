    GroundPlane

    GroundPlane is the shared lower infrastructure workspace for the nshkr platform core.

    It holds the reusable lower primitives that sit underneath outer_brain, Citadel, jido_integration, and app_kit.

    Scope

    • shared contracts and state vocabulary
    • Postgres transaction, outbox, inbox, and checkpoint helpers
    • generic projection publication helpers

    Internal Libraries

    • ground_plane_contracts
    • ground_plane_postgres
    • ground_plane_projection

    Status

    Workspace root established. The internal packages are intentionally small and generic.

    Development

    The project targets Elixir ~> 1.19 and Erlang/OTP 28.

    mix ci

    The welded ground_plane_contracts artifact is tracked through the prepared bundle flow:

    mix release.prepare
mix release.track
mix release.archive

    mix release.track updates the orphan-backed projection/ground_plane_contracts branch so downstream repos can pin a real generated-source ref before any formal release boundary exists.

    Workspace docs cover the overview, layout, contracts, Postgres helpers, and projection helpers.
